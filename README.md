# 🚀 Day 01 — Digital Clock (JavaScript Challenge)

This project is part of my **21 Days JavaScript Challenge** where I build real-world frontend projects using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

A real-time **Digital Clock** that updates every second and displays current time in 12-hour format with AM/PM along with the date.

---

## 🎯 Goal

- Practice JavaScript fundamentals  
- Understand real-time updates using `setInterval()`  
- Build clean and responsive UI  

---

## 🧩 Features

- ⏰ Live Time Update (Every Second)  
- 🕐 12-Hour Format with AM/PM  
- 📅 Current Date Display  
- 🎨 Clean & Centered UI  
- 📱 Responsive Design  

---

## 🧠 Flow (Planning Before Coding)

1. Get current date and time using `new Date()`  
2. Extract hours, minutes, seconds  
3. Convert time into 12-hour format  
4. Add AM/PM logic  
5. Format values (leading zero)  
6. Display time on UI  
7. Use `setInterval()` to update every second
